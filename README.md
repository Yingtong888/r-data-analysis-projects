# R Data Analysis Projects

A growing collection of R projects for data analysis, statistics, 
and visualization using the tidyverse.

## Dissertation Practice Series
| File | Description | Key Concepts |
|---|---|---|
| `Practice1.R` | Football player career statistics | Reshaping, line plots |
| `Practice2.R` | Premier League historical analysis | t-test, ANOVA, Tukey HSD |
| `Practice3.R` | GDP, CO2, life expectancy analysis | Joins, correlation, regression |
